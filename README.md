# SIKMUBYNCH (식무변처)

무한 웨이브를 막아내는 하이브리드 디펜스 게임. 스타크래프트 아이소메트릭 3D 뷰.

## 컨셉

- **하이브리드 디펜스**: 건물 배치(벙커) + 유닛 생산(마린) 복합 방어
- **무한 웨이브**: 점점 강해지는 적 웨이브를 얼마나 오래 버티는가
- **캐스캐이딩 실패**: 방어선이 뚫리면 연쇄 붕괴 — 긴장감의 핵심
- **스타크래프트 뷰**: 아이소메트릭 3D, RTS 스타일 조작

## 기술 스택

- HTML + Three.js (웹 브라우저에서 즉시 플레이)
- 싱글 HTML 파일 프로토타입

## 실행

`index.html`을 브라우저에서 열면 바로 플레이 가능.

## 구현된 기능

- **건물**: 커맨드센터(본진), 벙커(자동 사격), 배럭(마린 생산), 서플라이(인구 확장)
- **아군 유닛**: 마린 (배럭에서 자동 생산, 적 자동 공격)
- **적 유닛**: 저글링(빠름), 히드라리스크(원거리), 울트라리스크(탱커)
- **웨이브 시스템**: 무한 웨이브, 점진적 난이도 상승, 웨이브 간 준비 시간
- **자원 관리**: 미네랄 (적 처치 보상 + 웨이브 클리어 보너스)
- **캐스캐이딩 실패**: 건물 파괴 시 방어선 붕괴 → 연쇄 압박
- **카메라**: WASD 이동, Q/E 회전, 마우스휠 줌

## 조작법

| 키 | 동작 |
|---|------|
| `1` / `2` / `3` | 벙커 / 배럭 / 서플라이 선택 |
| `클릭` | 건물 배치 |
| `ESC` | 선택 취소 |
| `WASD` | 카메라 이동 |
| `Q` / `E` | 카메라 회전 |
| `마우스휠` | 줌 인/아웃 |
