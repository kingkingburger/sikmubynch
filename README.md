# SIKMUBYNCH (식무변처)

무한 웨이브를 막아내는 하이브리드 디펜스 게임. 스타크래프트 아이소메트릭 3D 뷰.

## 컨셉

- **하이브리드 디펜스**: 건물 배치(벙커) + 유닛 생산(마린) 복합 방어
- **무한 웨이브**: 점점 강해지는 적 웨이브를 얼마나 오래 버티는가
- **캐스캐이딩 실패**: 방어선이 뚫리면 연쇄 붕괴 — 긴장감의 핵심
- **스타크래프트 뷰**: 아이소메트릭 3D, RTS 스타일 조작

## 기술 스택

- HTML + Three.js (웹 브라우저에서 즉시 플레이)
- 싱글 HTML 파일 프로토타입

## 실행

`index.html`을 브라우저에서 열면 바로 플레이 가능.

## 상태

초기 프로토타입 — 아이디어 구체화 단계
