# SIKMUBYNCH (식무변처)

대규모 웨이브 디펜스 + 오토배틀 + 로그라이크. 3D 이소메트릭 게임.

## 컨셉

- **대규모 전투**: 첫 웨이브부터 수십 마리가 사방에서 밀려옴. 후반엔 500+
- **실시간 건설+전투**: They Are Billions 스타일. 웨이브 중에도 건설 가능
- **시너지 시스템**: 유닛+건물에 특성 부여, 조합으로 시너지 발동 (롤토체스식)
- **로그라이크**: 퍼마데스, 랜덤 맵, 웨이브 클리어 보상 3택
- **타격감**: 화면 흔들림, 히트스톱, 파티클. 대량 처치 시 쾌감

## 스크린샷

> 3D 이소메트릭 뷰 (디아블로/스타크래프트 스타일) — 플레이스홀더 메시 사용 중

## 기술 스택

- 3D 이소메트릭 (정사영 카메라)
- PC 전용 (키보드 + 마우스)

## 현재 상태

7-마일스톤 로드맵 (M1~M7) 중 M2 진행 중.

### M1: 코어 루프 ✅
- 64x64 타일 맵, 3D 이소메트릭 카메라
- 본진 (HQ) + 바리케이드 건설
- 적 스폰 (러셔 45마리), 직선 이동 + 건물 공격
- 미네랄 경제, 게임오버 + 재시작

### M2: 전투 기반 🔧
- 타워 (DPS 15, 사거리 6타일, 자동 공격) + 발사체
- 건물 레벨업 (Lv1→3, +30%/레벨)
- 철거 시스템 (50% 환불)
- 멀티웨이브 (적 스케일링 + 클리어 보너스)
- 건물 선택 핫바 (숫자키 1~2)

### M3~M7: 예정
- M3: 유닛 시스템 (배럭 + 4종 유닛 + 자동전투 AI)
- M4: 웨이브 + 적 다양성 (Flow Field, MultiMesh, 6종 적)
- M5: 메타 시스템 (시너지, 보상 3택, 이벤트)
- M6: 게임 필 (타격감, 카메라, UI, 사운드, 맵 생성)
- M7: 폴리시 (밸런싱, 최적화, 에셋 교체)

## 실행

에디터에서 `project/` 폴더 열기 → F5 실행.

## 조작

| 키 | 동작 |
|----|------|
| 1, 2 | 건물 선택 (바리케이드, 타워) |
| 좌클릭 (빈 타일) | 건물 배치 |
| 좌클릭 (기존 건물) | 레벨업 |
| 우클릭 (기존 건물) | 철거 (50% 환불) |

## 프로젝트 구조

```
project/
├── autoloads/          # 싱글톤 (GameManager)
├── scenes/
│   ├── main/           # 메인 게임 씬
│   ├── buildings/      # 건물 (HQ, Barricade, Tower)
│   ├── enemies/        # 적 유닛
│   └── projectiles/    # 발사체
├── scripts/data/       # 데이터 클래스 (Resource)
└── assets/             # 에셋
docs/ouroboros/         # Ouroboros 프로세스 문서
```
