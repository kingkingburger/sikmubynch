SIKMUBYNCH — Visual Style Comparison

A. 매트 로우폴리 (Hades 스타일)

날카로운 면 처리, 무광택, 색면 대비 강한 스타일라이즈드
무광택 + 선명한 색상 대비
+ Godot에서 바로 가능, 단색 머티리얼만으로 멋있음
- 텍스처 없이 밋밋할 수 있음

B. 리얼리스틱 다크 (Diablo 4 스타일)

사실적 그라데이션, 안개 효과, 빛과 그림자 대비
PBR 질감 + 안개 + 드라마틱 조명
+ 가장 고급스러운 느낌, 분위기 최고
- 텍스처가 있는 모델 필요 (AI 생성 또는 수동)

C. 픽셀아트 다크 (The Last Spell)

고해상도 도트 아트 캐릭터, 3D 지형 위 2D 빌보드
픽셀아트 + 네온 발광 + 어두운 톤
+ AI 이미지 생성으로 고퀄 스프라이트 가능
- 3D→2D 전환 작업 필요

D. 톤 셰이딩 (Polytopia 스타일)

플랫 색상 + 셀 셰이딩(2-3단계 그림자), 깔끔한 윤곽
셀 셰이딩 + 두꺼운 윤곽선 + 깔끔
+ 현재 3D 유지, 커스텀 셰이더로 구현 가능
- 다크 판타지 느낌 약할 수 있음
