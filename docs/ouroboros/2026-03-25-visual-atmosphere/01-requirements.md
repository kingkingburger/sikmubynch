# Phase 1: 요구사항 — 디아블로급 어두운 분위기 + 조명

## 목표 (Goal)

맵 전체를 어둡게 만들고, **발광체(건물/마법/적 눈)만 빛나는** "거점 등불" 스타일의 다크 판타지 분위기 구현. 디아블로 시리즈의 어두운 던전 느낌.

**한 문장:** "어둠 속에서 자기 거점만 빛나고, 적은 어둠에서 밀려오는 공포감"

## 핵심 엔티티

| 엔티티 | 발광 여부 | 설명 |
|--------|----------|------|
| HQ | 강한 발광 | 파란 마법 룬, 넓은 조명 반경 |
| Tower | 중간 발광 | 포탑 화염, 발사 시 번쩍 |
| Barracks | 약한 발광 | 내부 횃불 빛 |
| Miner | 중간 발광 | 청록 크리스탈 빛 |
| Buff Tower | 강한 발광 | 금색 크리스탈 오라 |
| 적 (일반) | 약한 발광 | 눈/신체 발광 (빨강/주황) |
| 적 (Exploder) | 강한 발광 | 전신 주황 펄스 |
| 적 (Destroyer) | 중간 발광 | 보라 오라 |
| 유닛 | 미약 발광 | 무기/갑옷 은은한 빛 |
| 발사체 | 강한 발광 | 비행 궤적 밝게 |
| 지면 | 비발광 | 거의 안 보임, 격자만 희미하게 |

## 제약조건

1. GL Compatibility 렌더러 — WorldEnvironment glow(bloom) 사용 가능
2. OmniLight3D/SpotLight3D 사용 가능 (그림자 포함)
3. 전체 화면 flash 금지 (사용자 눈 아픔 피드백)
4. 성능: 1000+ 적 + 동적 조명에서 60fps 유지
5. 모든 건물에 OmniLight3D 추가 시 성능 부하 → 건물당 1개 제한, 적은 발광 머티리얼만

## 비목표

1. UI 개선 (다음 단계)
2. 모델 퀄리티 업그레이드 (별도)
3. 사운드 변경

## 수용 기준

| # | 기준 |
|---|------|
| AC1 | 환경 조명(ambient) 최소화, 발광체 없는 영역은 거의 안 보임 |
| AC2 | 건물마다 OmniLight3D로 주변 밝힘 (HQ 반경 12, 타워 반경 6) |
| AC3 | WorldEnvironment glow(bloom) 적용 — 발광체가 빛번짐 |
| AC4 | 지면 셰이더: 기본 색상 더 어둡게, 비네트 강화 |
| AC5 | 적 emission 강화 — 어둠 속에서 빨간/주황 빛으로 보임 |
| AC6 | 발사체/이펙트 emission 강화 — 전투 시 빛의 축제 |
| AC7 | 60fps 유지 (F3 확인) |
