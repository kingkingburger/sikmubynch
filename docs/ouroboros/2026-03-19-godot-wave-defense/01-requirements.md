# SIKMUBYNCH (식무변처) - 요구사항

> Ouroboros Phase 1 | 2026-03-19 | Clarify 4차 보강 완료 (94개 결정 반영)

## 목표 (Goal)

**They Are Billions 스타일 실시간 웨이브 디펜스 + 오토배틀 + 로그라이크.**

플레이어가 실시간으로 건물을 짓고 유닛을 생산하며 무한 웨이브를 버티는 2D 탑다운 로그라이크 디펜스 게임. 유닛과 건물의 특성 조합으로 **다단계 시너지**가 발동하며(롤토체스식), 매 런마다 랜덤 맵/이벤트/보상으로 다른 경험을 제공한다.

### 핵심 경험 가치

1. **타격감** — 적 처치 시 화면 흔들림, 히트스톱, 이펙트. 대량 처치 시 쾌감. 정예 적 처치 시 강한 성취감.
2. **"한 판 더"** — 죽었을 때 바로 다시 시작하고 싶은 충동. "이번엔 이 시너지 Tier 3까지 가봐야지."
3. **1웨이브도 힘든 긴장감** — 0초부터 적 등장. 안전한 순간이 없음.
4. **대규모 스펙터클** — 첫 웨이브부터 압도적 물량. 500+ 동시 적.
5. **공포의 정예** — 정예 적 등장 시 방어선이 뚫리는 공포감. 대비하지 않으면 즉사급.

## 핵심 메커니즘

### 코어 루프

```
런 시작 (랜덤 맵)
  본진(중앙) + 바리케이드/타워(기본 2종) + 100 미네랄
  → 즉시 적 등장 (준비 시간 0초)
  → 실시간 건설 + 전투 동시 진행
    → 웨이브 클리어 시 보상 3택 (게임 일시정지 — 유일한 숨쉬기)
    → 즉시 다음 웨이브 (휴식 없음)
    → 반복...
  → 사망 (본진 파괴)
→ 게임오버 (웨이브 수/처치 수/시간)
→ 즉시 재시작
```

### 런 시작 상태

| 항목 | 값 |
|------|-----|
| 본진 (HQ) | 맵 중앙 고정. HP 1000. **느린 자동 회복 (초당 1 HP)**. 파괴 시 게임오버 |
| 시작 건물 | 바리케이드 + 타워 (기본 2종만 사용 가능) |
| 시작 자원 | 100 미네랄 |
| 해금 건물 | 배럭, 채굴기, 버프 타워 → 보상으로 해금 |
| 유닛 | 없음 → 배럭 해금+건설 후 자동 생산 시작 |

### 런 타임라인 (30분+ 목표)

| 시점 | 상태 | 웨이브 |
|------|------|--------|
| 0초 | 런 시작. **즉시 러셔 45마리 등장** | Wave 1 (잔잔) |
| ~1분 | 첫 웨이브 클리어 → 보상 3택 | Wave 2 (잔잔) |
| ~3분 | 배럭 해금 시도. 유닛 생산 시작 | Wave 3 (폭풍!) |
| ~5분 | 첫 정예 적 등장. 방어선 위기 | Wave 5 (폭풍!) |
| ~10분 | 시너지 Tier 1~2 활성. 100+ 동시 적 | Wave 8-10 |
| ~20분 | 시너지 Tier 3 도달 가능. 300+ 동시 적 | Wave 15-20 |
| ~30분+ | 극한 서바이벌. 500+ 동시 적. 정예 다수 | Wave 25+ |

### 페이스 (Pacing)

- **준비 시간 0초** — 런 시작 즉시 적 등장
- **웨이브 간 휴식 0초** — 웨이브 연속 진행
- **유일한 숨쉬기** — 보상 3택 선택 화면 (게임 일시정지)
- **런 길이 30분+** — 죽으면 타격 크지만 "한 판 더" 하게 됨

## 건물 상세 (6종)

### 시작 건물 (처음부터 사용 가능)

| 건물 | 비용 | HP | 크기 | 역할 | 특성 | 상세 |
|------|------|-----|------|------|------|------|
| **본진 (HQ)** | - | 1000 | 3x3 | 방어 대상 | - | 파괴 시 게임오버. 약한 자동 공격 (DPS 5, 사거리 3). **느린 자동 회복 (초당 1 HP)**. 최후의 보루 |
| **바리케이드** | 15 | 50 | 1x1 | 시간벌이 벽 | 가변 | **종이벽**. 싸고 약함. 대량 배치하여 적 이동 차단. 부서지면 다시 짓기. 드래그로 자유 드로잉 배치 |
| **타워** | 60 | 150 | 1x1 | 원거리 공격 | 가변 | 자동 사격. DPS 15, 사거리 6. 핵심 딜러 건물 |

### 해금 건물 (보상으로 획득)

| 건물 | 비용 | HP | 크기 | 역할 | 특성 | 상세 |
|------|------|-----|------|------|------|------|
| **배럭** | 80 | 120 | 2x1 | 유닛 생산 | 가변 | 유닛 자동 생산 (8초마다 1마리). 솔저/아처/탱커/폭탄병 **완전 랜덤** 생산. 특성이 생산 유닛에 전달 |
| **채굴기** | 50 | 80 | 1x1 | 경제 | - | **미네랄 노드 위에만** 배치 가능. 초당 2 미네랄 자동 생산 |
| **버프 타워** | 100 | 100 | 1x1 | 보조 | 가변 | **좌우 3타일 범위 (7x7 영역)** 내 건물/유닛 강화. DPS +20%. 시너지 트리거 핵심. **중첩 가능** (버프타워 2개 범위 겹치면 DPS +40%) |

> **"가변"** = 보상으로 획득 시 랜덤 특성 부여. 같은 건물이라도 특성이 다를 수 있음.

### 건물 레벨업 시스템

| 항목 | 내용 |
|------|------|
| 최대 레벨 | Lv1 → Lv2 → Lv3 |
| 레벨업 방식 | 미네랄 소모 |
| 레벨당 효과 | **전체 스탯 +30%** (HP, DPS, 사거리, 생산속도 등) |
| Lv2 비용 | 건설 비용의 100% |
| Lv3 비용 | 건설 비용의 200% |
| 레벨업 시각 | **스프라이트 변경** (레벨별 다른 외형. Lv1→Lv2→Lv3 점점 화려해짐) |

| 건물 | 건설 비용 | Lv2 비용 | Lv3 비용 |
|------|----------|---------|---------|
| 바리케이드 | 15 | 15 | 30 |
| 타워 | 60 | 60 | 120 |
| 배럭 | 80 | 80 | 160 |
| 채굴기 | 50 | 50 | 100 |
| 버프 타워 | 100 | 100 | 200 |

### 건물 철거 / 파괴 / 잔해

| 항목 | 내용 |
|------|------|
| 철거 | 가능. **50% 환불** |
| 수리 | **불가**. 파괴되면 끝, 새로 짓기 |
| 파괴 연출 | **폭발 이펙트 + 잔해 남음** |
| 잔해 | 잔해 위에 **즉시 재건설 가능** (추가 비용 없음) |
| 보상 중복 | **허용**. 같은 건물 여러 개 배치 가능 |

### 건물 배치 규칙

| 항목 | 내용 |
|------|------|
| 건물 간격 | **없음** — 바로 옆에 붙여서 배치 가능 |
| 그리드 스냅 | 모든 건물은 타일 그리드에 스냅 |
| 채굴기 제한 | 미네랄 노드 위에만 배치 가능 |

### 바리케이드 상세 (핵심 건물)

- **종이벽 철학**: 싸고(15) 약하고(HP 50) 많이 짓는다
- **자유 드로잉 배치**: 마우스 경로를 따라 벽이 자유롭게 배치됨 (바리케이드 전용, 직선 아닌 곡선도 가능)
- **드래그 최대 길이**: **미네랄이 허락하는 만큼** (돈 떨어지면 자동 멈춤)
- **시간벌이**: 적이 부수는 데 ~3초. 그 사이에 타워/유닛이 공격
- **특성 바리케이드**: 화염 바리케이드 = 접촉 시 화염 데미지. 냉기 = 감속 등
- **전략**: 다층 방어선 (1차 벽 → 타워 → 2차 벽 → 본진)

### 건물이 장애물

- **모든 건물이 적 경로의 장애물** 역할
- 적은 건물을 **부수고 지나갈 수 있음** (통과는 불가)
- 건물 배치 자체가 미로/경로 설계

## 유닛 상세 (4종, 완전 오토)

모든 유닛은 **완전 자동 전투**. 플레이어는 명령 불가. 가장 가까운 적 자동 공격.

| 유닛 | 생산 건물 | HP | DPS | 사거리 | 속도 | 특성 | 상세 |
|------|----------|-----|-----|--------|------|------|------|
| **솔저** | 배럭 | 80 | 10 | 근접(1) | 2.5 | 배럭 계승 | 기본 근접. 적에게 돌진. 체력이 높아 벽 역할도 |
| **아처** | 배럭 | 40 | 12 | 원거리(5) | 2.0 | 배럭 계승 | 원거리 딜러. 유리몸. 타워 뒤에 배치하면 효과적 |
| **탱커** | 배럭 | 200 | 5 | 근접(1) | 1.5 | 강화 고정 | 높은 체력. 적 어그로 끌기. 벽 앞에서 적 묶기 |
| **폭탄병** | 배럭 | 30 | 범위 25 | 근접(2) | 2.0 | 화염 고정 | 적에게 돌진 후 자폭. 범위 데미지. 소모형 유닛. **아군 피해 없음** (적만 데미지) |

> 생산 시간: 8초마다 1마리. 배럭 1개 = 분당 7.5마리

### 유닛 행동 규칙

| 항목 | 내용 |
|------|------|
| 이동 방식 | **적극적 순찰** — 맵을 돌아다니며 적을 찾음 |
| 사망 처리 | **소모품** — 죽으면 끝, 배럭이 새로 생산 |
| 인구 제한 | **없음** — 무한 생산 가능 |
| 생산 선택 | **불가** — 배럭이 솔저/아처/탱커/폭탄병을 **완전 랜덤** 생산 (매번 4종 중 랜덤 선택) |
| 특성 전달 | 배럭의 특성이 생산된 유닛에 **그대로 전달** |
| 다중 배럭 | 각 배럭이 **독립적**으로 자기 특성의 유닛 생산 (화염 배럭→화염 유닛, 냉기 배럭→냉기 유닛) |

### 유닛 없을 때

- 유닛이 없으면 적은 **건물만 공격**
- 공격 우선순위: **바리케이드 → 타워 → 본진** 순서

## 적 상세 (4종 + 정예)

### 적 행동 규칙

| 항목 | 내용 |
|------|------|
| 타격 우선순위 | **가장 가까운 것** 공격 (건물이든 유닛이든 구분 없음) |
| 이동 시간 | 방어 없이 맵 가장자리 → 본진 **약 10초** (빠름) |
| 경로 | 모든 건물이 장애물. 적은 건물을 **부수고 지나갈 수 있음** |
| 공격 연출 | **때리는 느낌** (물리 타격). 타격 이펙트 + 건물 흔들림 |
| 사망 시 | **미네랄 오브가 바닥에 떨어짐** → 본진으로 물리적으로 날아가서 획득 (~2초 소요. 멀리서 죽으면 늦게 도착) |
| 시각 구분 | **색상 + 크기 모두 다름** (러셔=작고 초록, 탱크=크고 빨강 등) |
| 정예 경로 | **바리케이드를 즉시 파괴**하며 전진 (다른 건물은 막힘) |
| 리타겟 | 건물/유닛 파괴 후 **즉시 다음 가까운 건물/유닛 공격** (바로 전환) |
| 맵 범위 | **맵 밖 불가** — 맵 경계에서 스폰, 맵 안에서만 이동 |

### 일반 적 (잡몹 — 물량이 위협)

| 적 | HP | DPS | 속도 | 보상 | 시각 | 특징 |
|----|-----|-----|------|------|------|------|
| **러셔** | 20 | 8 | 3.5 | 3 미네랄 | 작고 초록 | 약하고 빠름. **대량 출현**. 바리케이드 1초에 부숨. 물량으로 압도 |
| **스플리터** | 60 | 10 | 2.5 | 8 미네랄 | 중간 보라 | 사망 시 2마리 미니 스플리터로 분열 (HP 20, 1회 한정). **분열체 보상 있음** (미니 스플리터당 2 미네랄) |
| **폭발형** | 40 | 범위 50 | 3.0 | 12 미네랄 | 중간 주황 | 건물 근접 시 자폭. 범위 데미지. 바리케이드 즉시 파괴 |

### 정예 적 (**공포감** — 간헐 등장, 방어선 뚫기)

| 적 | HP | DPS | 속도 | 보상 | 시각 | 특징 |
|----|-----|-----|------|------|------|------|
| **탱크** | 300 | 25 | 1.5 | 20 미네랄 | 크고 빨강 | 높은 체력. **바리케이드 즉시 파괴**하며 전진 (다른 건물은 막힘). 타워 직접 공격 |
| **엘리트 러셔** | 100 | 15 | 4.0 | 15 미네랄 | 중간 진초록 | 빠르고 강한 러셔. 바리케이드 즉시 파괴. 5웨이브부터 등장 |
| **파괴자** | 500 | 40 | 1.0 | 40 미네랄 | 매우 크고 검정 | **보스급**. 모든 건물을 2-3초에 부숨. 10웨이브부터 등장. 등장 시 경고 이펙트 |

> 정예 적 등장 시 화면 가장자리 경고 + 특수 사운드 + 적 강조 표시

### 적 위협 모델

```
일반 웨이브: 러셔 40마리 + 스플리터 5마리 = "개미 떼"
폭풍 웨이브: 러셔 80마리 + 스플리터 15 + 폭발형 10 + 탱크 3 = "지옥"
보스 웨이브: 위 + 파괴자 1마리 = "방어선 뚫림 공포"
```

## 웨이브 시스템 (파도형 난이도)

### 웨이브 규칙

| 항목 | 내용 |
|------|------|
| 웨이브 길이 | **45초** |
| 웨이브 클리어 | **시간 기반** — 45초마다 다음 웨이브 시작 (적이 남아있어도) |
| 30분 기준 | 약 **40웨이브** |

### 파도형 난이도 곡선

```
잔잔 → 잔잔 → 폭풍! → 잔잔 → 잔잔 → 폭풍! → ...

잔잔 웨이브: 러셔 중심. 자원 벌이 + 건설 타이밍
폭풍 웨이브: 대량 + 정예 + 이벤트. 생존 테스트
```

| 웨이브 타입 | 패턴 | 적 규모 | 정예 | 이벤트 |
|------------|------|---------|------|--------|
| 잔잔 | 1, 2, 4, 5, 7, 8, ... | 기본 x 0.7 | 없음 | 50% 확률 |
| **폭풍** | 3, 6, 9, 12, 15, ... | 기본 x 1.5 | 있음 | 100% |

### 체력/속도 스케일링

```
Wave N의 스케일링:
  체력 배수: 1.0 + (N-1) x 0.08  (Wave 10: x1.72, Wave 20: x2.52)
  속도 배수: 1.0 + (N-1) x 0.015 (Wave 10: x1.14, Wave 20: x1.29)
```

### 스폰 연출

- 사방 맵 가장자리에서 랜덤 위치 스폰
- 초당 최대 50마리 순차 스폰 (물결처럼 밀려오는 연출)
- 폭풍 웨이브: 특정 방향에서 집중 스폰 가능
- 정예 적: 등장 2초 전 경고 이펙트 (빨간 화살표 + 경고음)

## 수용 기준 (Acceptance Criteria)

### 타격감

- [ ] 러셔 처치 시 white flash + 작은 파티클
- [ ] 대량(10마리+) 동시 처치 시 강한 화면 흔들림 + 쾌감
- [ ] 정예 적 처치 시 히트스톱(100ms) + 대형 폭발 이펙트
- [ ] 파괴자 처치 시 화면 전체 흔들림 + 슬로모션 연출
- [ ] 적 공격이 물리 타격 느낌 (타격 이펙트 + 건물 흔들림)
- [ ] **화면 흔들림 제한 없음** (대량 처치 시 극한 쾌감)
- [ ] **크리티컬 히트** (10%, 2배 데미지, 큰 숫자 + 특수 이펙트 표시)

### 긴장감

- [ ] 0초에 적이 밀려오며 즉시 판단/건설이 필요함
- [ ] 정예 적 경고 시 "저거 막을 수 있을까?" 공포감
- [ ] 폭풍 웨이브에서 방어선이 뚫리는 위기감
- [ ] 30분+ 플레이 중 지루한 순간이 없음 (파도형 난이도)
- [ ] 적 남아있어도 45초 후 다음 웨이브 시작 → 누적 압박감

### 시너지

- [ ] Tier 1 달성 시 "있으면 좋네" 체감
- [ ] Tier 2 달성 시 "오 달라졌다" 체감 + 전략 변경 유발
- [ ] Tier 3 달성 시 "이걸로 이겼다" 판 뒤집기 체감
- [ ] 시너지 바(좌측 세로)가 직관적 (아이콘 + 단계 + 진행도)
- [ ] 보상 카드에 시너지 영향 표시 ("화염 Tier 2 달성!")

### 리플레이

- [ ] 죽었을 때 "한 판 더" 하고 싶은 충동
- [ ] 시너지 빌드 경로가 매 런 다름
- [ ] 랜덤 이벤트로 같은 전략이 매번 통하지 않음
- [ ] 게임오버에서 1클릭 재시작 가능

### 유닛

- [ ] 유닛이 맵을 돌아다니며 적극적으로 적 탐색
- [ ] 유닛 사망 시 자연스럽게 소멸 (소모품 체감)
- [ ] 배럭이 4종 유닛 완전 랜덤 생산
- [ ] 배럭 특성이 생산된 유닛에 전달

### 건물

- [ ] 건물 레벨업 (Lv1→Lv2→Lv3) 정상 동작
- [ ] 건물 철거 시 50% 환불
- [ ] 건물 파괴 시 잔해 남음 + 잔해 위 즉시 재건설 가능
- [ ] 채굴기는 미네랄 노드 위에만 배치 가능

### 시스템

- [ ] 실시간 건설+전투 동시 진행
- [ ] 바리케이드 자유 드로잉 배치 정상 동작
- [ ] 유닛 완전 자동 전투 (명령 불필요)
- [ ] 보상 3택 선택 즉시 적용
- [ ] **보상 스킵 가능** (페널티 없음)
- [ ] 이벤트가 거의 매 웨이브 발생
- [ ] **선택형 이벤트가 전투 중 표시** (일시정지 없음. 거절 불가, A/B 필수 선택)
- [ ] 파도형 난이도 (잔잔 → 폭풍 반복) 체감
- [ ] 카메라 WASD 스크롤 + 줌 + 미니맵 정상
- [ ] **일시정지 중 모든 조작 가능** (건설 + 정보 확인 + 보상 + 이벤트 선택)
- [ ] Alt 누르는 동안 HP 바 표시
- [ ] 미네랄 오브 떨어짐 → 본진 흡수 연출
- [ ] 게임 속도 F 키로 1x/2x/3x 전환
- [ ] **본진 HP 초당 1 회복**
- [ ] **본진 공격 시 빨간 비네트 경고**

### 사운드

- [ ] 9종 효과음 모두 재생
- [ ] 타격음, 사망음, 배치음이 게임 리듬감에 기여

### 성능

- [ ] 500적 60fps
- [ ] 1000적 30fps
- [ ] 30분+ 메모리 누수 없음
