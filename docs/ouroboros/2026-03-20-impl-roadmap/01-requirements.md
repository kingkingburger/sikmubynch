# SIKMUBYNCH - 구현 로드맵 요구사항

> Ouroboros Phase 1 | 2026-03-20 | 모호성: 20%

## 목표 (Goal)

Ouroboros 3Phase에서 확정된 **94개 결정**을 Godot 4로 실제 구현하는 **7단계 로드맵**. 각 마일스톤은 플레이 가능한 빌드를 산출하며, 직접 플레이 + 피드백으로 완료를 판단한다.

## 7단계 마일스톤

### M1: 코어 루프

**목표**: "게임이 돈다"는 최소 증명. 색상 도형만으로.

| 항목 | 내용 |
|------|------|
| 맵 | 64x64 그리드 (색상 타일) |
| 본진 | 중앙 3x3 (HP 1000, 파괴 시 게임오버) |
| 바리케이드 | 클릭 배치 (비용 15, HP 50, 1x1) |
| 적 | 러셔 45마리 스폰 (가장자리에서 본진으로 이동) |
| 전투 | 적이 건물 공격 → 건물 파괴 → 본진 도달 |
| 게임오버 | 본진 HP 0 → 결과 표시 + 재시작 |
| 자원 | 적 처치 시 미네랄 획득 (기본) |
| 아트 | 플레이스홀더 (색상 사각형/원) |

**아직 없는 것**: 타워, 유닛, 시너지, 보상, 웨이브, 타격감, UI, 카메라, 미니맵

---

### M2: 전투 기반

**목표**: 건설 + 사격 전투가 동작하는 기본 디펜스.

| 항목 | 내용 |
|------|------|
| 타워 | 자동 사격 (DPS 15, 사거리 6). 발사체 시각화 |
| 미네랄 경제 | 적 처치 보상 + 웨이브 클리어 보너스. 건설 비용 차감 |
| 발사체 | 눈에 보이는 projectile (타워→적) |
| 기본 UI | 상단 HUD (미네랄, 웨이브 수) + 하단 건설 메뉴 (숫자키 1~3) |
| 건물 레벨업 | Lv1→Lv2→Lv3 (미네랄 소모, 스탯 +30%) |
| 건물 철거 | 우클릭 → 50% 환불 |

**새로 추가**: 타워, 발사체, 경제, 기본 HUD, 레벨업, 철거

---

### M3: 유닛 시스템

**목표**: 오토배틀 유닛이 전장에서 싸우는 하이브리드 디펜스.

| 항목 | 내용 |
|------|------|
| 배럭 | 유닛 자동 생산 (8초마다 1마리, 4종 랜덤) |
| 솔저 | 근접 전투 (HP 80, DPS 10) |
| 아처 | 원거리 공격 (HP 40, DPS 12, 사거리 5) |
| 탱커 | 높은 체력 (HP 200, DPS 5) |
| 폭탄병 | 자폭 범위 공격 (HP 30, DPS 25, 아군 피해 없음) |
| 유닛 AI | 적극적 순찰 — 맵을 돌며 적 탐색, 가장 가까운 적 자동 공격 |
| 사망 처리 | 소모품 — 죽으면 끝, 배럭이 새로 생산 |

**새로 추가**: 배럭, 유닛 4종, 자동전투 AI

---

### M4: 웨이브 + 적 다양성

**목표**: 파도형 난이도와 다양한 적으로 긴장감 있는 서바이벌.

| 항목 | 내용 |
|------|------|
| 웨이브 시스템 | 45초 간격, 시간 기반 진행 (적 남아도 다음 웨이브) |
| 파도형 난이도 | 잔잔→잔잔→폭풍 반복 |
| 적 4종 | 러셔, 스플리터(분열), 폭발형(자폭), 탱크(정예) |
| 정예 적 | 엘리트 러셔, 파괴자 (등장 시 경고) |
| 스케일링 | 웨이브별 적 수/체력/속도 증가 |
| Flow Field | BFS 기반 길찾기 (건물 = 장애물) |
| 적 사망 | 미네랄 오브 드롭 → 본진으로 비행 (~2초) |

**새로 추가**: 웨이브, 적 6종, 스케일링, Flow Field, 미네랄 오브

---

### M5: 메타 시스템

**목표**: 시너지/보상/이벤트로 "한 판 더" 리플레이 가치 확보.

| 항목 | 내용 |
|------|------|
| 특성 5종 | 화염, 냉기, 독, 전기, 강화 |
| 시너지 3단계 | Tier 1(2개 +20%), Tier 2(4개 +50%+특수), Tier 3(6개 +100%+판뒤집기) |
| 크로스 시너지 | 원소 마스터, 원소 군주 |
| 보상 3택 | 웨이브 클리어 시 3택 (건물/유닛/버프). 희귀도 3단계. 스킵 가능 |
| 이벤트 | 전투 변수 5종(3초 예고) + 선택형 5종(전투 중, 거절 불가) |
| 채굴기 | 미네랄 노드 위에만 배치, 초당 2 미네랄 |
| 버프 타워 | 좌우 3타일 범위 내 DPS +20%, 중첩 가능 |

**새로 추가**: 시너지 전체, 보상, 이벤트, 채굴기, 버프타워

---

### M6: 게임 필

**목표**: 타격감 + 연출 + UI로 "기분 좋은 게임" 완성.

| 항목 | 내용 |
|------|------|
| 타격감 | 화면 흔들림(제한 없음) + 히트스톱 + 파티클 + white flash |
| 크리티컬 히트 | 10% 확률, 2배 데미지, 큰 숫자 + 특수 이펙트 |
| 카메라 | WASD 스크롤 + 마우스 휠 줌 (0.5x~2x) |
| 미니맵 | 우하단, 클릭 이동 |
| 시너지 바 | 좌측 세로 (롤토체스 스타일) |
| 게임오버 UI | 웨이브 수 + 처치 수 + 시간 + 즉시 재시작 |
| 사운드 | 9종 효과음 |
| 절차적 맵 생성 | 미네랄 노드/장애물/통로 랜덤 배치 + 경로 유효성 검증 |
| HP 표시 | Alt 토글 |
| 일시정지 | Space (모든 조작 가능) |
| 게임 속도 | F 키 (1x/2x/3x) |

**새로 추가**: 타격감, 카메라, 풀 UI, 사운드, 맵생성, 조작 전체

---

### M7: 폴리시

**목표**: 밸런싱 + 성능 최적화 + 버그 수정으로 출시 품질 달성.

| 항목 | 내용 |
|------|------|
| 밸런싱 | 수치 튜닝 (DPS, HP, 비용, 시너지 효과, 웨이브 스케일링) |
| 성능 | MultiMesh 적용 (500적 60fps, 1000적 30fps), Object Pool, 메모리 최적화 |
| 건물 레벨업 시각 | 레벨별 스프라이트 변경 |
| 픽셀아트 | 플레이스홀더 → 실제 에셋 교체 |
| 버그 수정 | 플레이테스트 기반 |
| 디버그 오버레이 | F3 (fps, 적 수, 풀 상태) |
| 시작 화면 | 타이틀 + "시작" 버튼 |
| 설정 메뉴 | ESC → 재시작 + 나가기 |

**새로 추가**: 밸런싱, 성능 최적화, 에셋 교체, 버그 수정

## 마일스톤 완료 프로세스

```
각 마일스톤 완료 시:
  1. 기능 체크리스트 전부 통과 확인
  2. 빌드 실행 → 직접 플레이
  3. "이게 맞나?" 주관적 판단
  4. 피드백 반영 (필요 시 조정)
  5. 다음 마일스톤 진행
```

## 제약조건 (Constraints)

| 항목 | 값 |
|------|-----|
| 엔진 | Godot 4 + GDScript |
| 아트 | M1~M6 플레이스홀더 → M7에서 픽셀아트 교체 |
| 성능 최적화 | M7에서 집중 (그 전에는 기능 우선) |
| 각 마일스톤 | 플레이 가능한 빌드 산출 |
| 기술 결정 | AI 자율 (게임 경험 관련만 사용자 확인) |

## 비목표 (Non-Goals)

- 메타 진행 / 영구 업그레이드 (MVP 이후)
- 멀티플레이 / 네트워크
- 스토리 / 캠페인
- 모바일 지원

## 수용 기준 (Acceptance Criteria)

- [ ] M1~M7 모든 마일스톤 체크리스트 통과
- [ ] 각 마일스톤에서 직접 플레이 + 피드백 완료
- [ ] 최종 빌드: 500적 60fps, 30분+ 메모리 누수 없음
- [ ] 03-verification.md의 게임 필 체크리스트 통과
