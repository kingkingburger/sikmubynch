# SIKMUBYNCH - 구현 로드맵 검증

> Ouroboros Phase 3 | 2026-03-20 | 모호성: 16%

## 검증 전략

**마일스톤별 빌드 + 직접 플레이 + 구두 피드백 + 회귀 테스트**

```
매 마일스톤 완료 프로세스:
  1. 기능 체크리스트 확인
  2. 전 마일스톤 회귀 테스트 (5분, 이전 기능 깨짐 확인)
  3. 빌드 실행 → 사용자 직접 플레이
  4. 구두 피드백 ("이건 좋았는데 저건 이상해")
  5. 피드백 반영 → 다음 마일스톤 진행
```

## M1: 코어 루프 — 검증 체크리스트

### 기능 체크리스트

- [ ] **맵**: 64x64 타일 그리드가 화면에 렌더링됨
- [ ] **본진**: 중앙 3x3 위치, HP 1000 표시 가능
- [ ] **본진 회복**: 초당 1 HP 자동 회복 동작
- [ ] **바리케이드 배치**: 클릭으로 1x1 바리케이드 배치 (그리드 스냅)
- [ ] **바리케이드 비용**: 15 미네랄 차감, 부족 시 배치 불가
- [ ] **적 스폰**: 맵 가장자리에서 러셔 45마리 등장
- [ ] **적 이동**: 본진 방향으로 직선 이동
- [ ] **적 건물 충돌**: 경로 상 건물 만나면 공격 → 파괴 → 전진
- [ ] **적 본진 공격**: 본진 도달 시 HP 감소
- [ ] **적 처치**: 적 HP 0 시 사라짐 + 미네랄 획득 (3)
- [ ] **게임오버**: 본진 HP 0 → 게임오버 화면
- [ ] **재시작**: 게임오버에서 1클릭 재시작
- [ ] **시작 자원**: 100 미네랄

### 플레이테스트 핵심 질문

- "바리케이드를 배치하면 적이 부수고 지나가는가?"
- "미네랄이 자연스럽게 순환하는가? (벌고 → 쓰고)"
- "게임이 '돌아가는' 느낌이 있는가?"

### 엣지케이스

| 상황 | 기대 동작 |
|------|----------|
| 미네랄 0에서 건설 시도 | 배치 불가 (시각적 피드백) |
| 건물 위에 건물 배치 | 배치 불가 |
| 적 45마리 동시 본진 도달 | 게임오버 정상 트리거 |
| 바리케이드로 적 완전 차단 | 적이 바리케이드 공격해서 부수고 전진 |

## M4: 웨이브 + 적 다양성 — 검증 체크리스트

- [ ] **웨이브 시스템**: 45초 간격 자동 진행

- [ ] **적 스케일링**: 웨이브별 적 수/체력/속도 증가
- [ ] **스플리터**: 사망 시 2마리 분열 (1회 한정)
- [ ] **폭발형**: 건물 근접 시 자폭 범위 공격
- [ ] **탱크**: 높은 HP, 바리케이드 즉시 파괴
- [ ] **엘리트 러셔**: 5웨이브부터 등장
- [ ] **파괴자**: 10웨이브부터 등장, 등장 경고
- [ ] **Flow Field**: 적이 건물을 우회하여 이동
- [ ] **Flow Field 갱신**: 건물 배치/파괴 시 경로 재계산
- [ ] **MultiMesh**: 200+ 적 동시 렌더링
- [ ] **적 남아도 다음 웨이브**: 45초 후 강제 진행

### 성능 기준

- [ ] 200적 동시: 60fps 이상
- [ ] Flow Field 재계산: < 16ms

- [ ] **크리티컬**: 10% 확률, 2배 데미지, 큰 숫자

- [ ] **마우스 휠 줌**: 0.5x~2x
