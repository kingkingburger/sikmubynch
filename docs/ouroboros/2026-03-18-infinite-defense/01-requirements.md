# 01-requirements.md — SIKMUBYNCH 식무변처

> **Status**: Phase 1 진행 중 (프로토타입 우선 진행)
> **모호성**: 63% (게이트 미통과, 프로토타입 기반 반복 예정)

## 목표 (Goal)

무한히 밀려오는 적 웨이브를 건물 배치 + 유닛 생산 하이브리드 방어로 막는 싱글플레이 디펜스 게임.
스타크래프트 아이소메트릭 뷰의 3D 웹 게임.

## 핵심 메커니즘

- **하이브리드 디펜스**: 고정 방어 건물(벙커) + 기동 유닛(마린) 복합 운영
- **웨이브 시스템**: 무한 웨이브, 점진적 난이도 상승
- **자원 관리**: 적 처치 보상 + 웨이브 클리어 보너스로 미네랄 획득 → 건물/유닛에 투자
- **캐스캐이딩 실패**: 방어선 붕괴 시 적이 쏟아져 들어오는 긴장감. "한번 뚫리면 계속 뚫린다"

## 패배 조건

- 커맨드센터(본진) 파괴 시 게임 오버
- 건물이 파괴되면 방어선에 구멍 → 적이 후방으로 밀고 들어옴 → 연쇄 붕괴

## 제약조건 (Constraints)

- **플랫폼**: 웹 (HTML + Three.js), 브라우저에서 즉시 플레이
- **플레이 모드**: 싱글플레이만
- **뷰**: 스타크래프트 스타일 아이소메트릭 3D

## 비목표 (Non-Goals)

- (미확정) 멀티플레이, 네트워크, 계정 시스템
- (미확정) 모바일 최적화
- (미확정) 스토리/캠페인

| 가정 | 상태 |
|------|------|

## 핵심 엔티티 (Ontology)

- **커맨드센터**: 본진, 파괴 시 게임오버
- **벙커**: 고정 방어 건물, 자동 사격
- **배럭**: 마린 자동 생산 건물
- **서플라이**: 최대 인구 확장
- **마린**: 기동 전투 유닛, 적 자동 공격
- **적 유닛**: 저글링(빠름), 히드라(원거리), 울트라(탱커)
- **미네랄**: 유일한 자원
