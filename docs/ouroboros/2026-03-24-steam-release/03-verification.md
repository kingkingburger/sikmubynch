## E2E 시나리오

### S-STEAM-1: GodotSteam 초기화

**전제 조건**: GodotSteam 플러그인 + 커스텀 Export Template 설치
**스텝**:
1. Steam 클라이언트 실행
2. 게임 빌드 실행
3. 콘솔/로그에서 "Steam initialized successfully" 확인
4. Steam 오버레이 (Shift+Tab) 동작 확인

**성공 판정**: 초기화 성공 + 오버레이 동작

### S-STEAM-2: 업적 동작

**전제 조건**: S-STEAM-1 통과, Steamworks에 업적 15개 등록
**스텝**:
1. 게임 시작 → 첫 킬 → "첫 처치" 업적 팝업 확인
2. 웨이브 5 도달 → "5번째 파도" 업적 팝업 확인
3. Steam 오버레이에서 업적 목록 확인
4. 게임 종료 후 Steam 프로필에서 업적 반영 확인

**성공 판정**: 업적 해제 + 팝업 + 프로필 반영

### S-STEAM-3: 리더보드

**전제 조건**: S-STEAM-1 통과, Steamworks에 리더보드 3개 등록
**스텝**:
1. 게임 플레이 → 게임 오버
2. 웨이브/킬/시간 스코어 업로드 확인 (로그)
3. Steam 오버레이에서 리더보드 조회

**성공 판정**: 스코어 업로드 + 리더보드 조회 가능

### S-STEAM-4: 클라우드 세이브

**전제 조건**: Steamworks Auto-Cloud 설정 완료
**스텝**:
1. 게임 3판 플레이 → 통계 누적
2. 게임 종료
3. stats.json이 Steam Cloud에 동기화되었는지 확인
4. (가능하면) 다른 PC에서 통계 동기화 확인

**성공 판정**: user://stats.json이 클라우드 동기화됨

### S-STEAM-5: 통계 시스템

**전제 조건**: StatsManager 구현 완료
**스텝**:
1. 게임 3판 플레이
2. 타이틀 화면에서 누적 통계 표시 확인
3. 최고 웨이브, 총 킬, 플레이 횟수 정확한지 확인
4. 게임 종료 후 재실행 → 통계 유지 확인

**성공 판정**: 통계 저장/로드/표시 정상

### S-STEAM-6: 사운드 시스템

**전제 조건**: AudioManager + 오디오 파일 준비
**스텝**:
1. 타이틀 화면 → BGM 재생 확인
2. 게임 시작 → 전투 BGM 전환 확인
3. 건물 배치 → 배치 효과음
4. 적 처치 → 사망 효과음
5. ESC 메뉴 → 볼륨 조절 → 즉시 반영 확인

**성공 판정**: BGM 2곡 + 핵심 SFX 5종 이상 정상 재생

### S-STEAM-7: 화면 설정

**전제 조건**: SettingsManager 구현 완료
**스텝**:
1. ESC 메뉴 → 설정
2. 풀스크린 ↔ 창모드 토글 → 즉시 반영
3. 해상도 변경 (720p → 1080p) → 반영
4. 게임 종료 후 재실행 → 설정 유지 확인

**성공 판정**: 설정 변경 + 저장 + 재실행 시 유지

### S-STEAM-8: 스토어 페이지

**전제 조건**: 모든 에셋 업로드
**스텝**:
1. Steamworks 파트너 사이트에서 스토어 페이지 미리보기
2. 캡슐 이미지 7종 정상 표시 확인
3. 스크린샷 5장+ 확인
4. 트레일러 재생 확인
5. 한글/영문 설명 모두 확인
6. 태그 5개 설정 확인

**성공 판정**: 모든 에셋 정상 표시 + 설명 완료

### S-STEAM-9: 빌드 안정성

**전제 조건**: Windows 빌드 업로드 완료
**스텝**:
1. Steam에서 게임 다운로드 + 설치
2. 실행 → 타이틀 화면 진입
3. 3판 연속 플레이 (각 10분+)
4. 게임 중 ESC → 타이틀 → 재시작 반복
5. 총 30분+ 연속 실행

**성공 판정**: 크래시/프리즈 0회, 메모리 누수 없음

### S-STEAM-10: 후원 DLC

**전제 조건**: Steamworks에 후원 DLC 등록
**스텝**:
1. 스토어 페이지에서 후원 DLC 표시 확인
2. DLC 설명에 "후원용, 게임 내 기능 없음" 명시 확인

**성공 판정**: DLC 페이지 정상 + 구매 가능

## 엣지케이스 시나리오

### E-STEAM-1: Steam 미실행 시 게임 실행

**상황**: Steam 클라이언트 없이 .exe 직접 실행
**기대 동작**: Steam 초기화 실패 → 게임은 정상 실행되되 Steam 기능 비활성
**검증**: Steam 종료 후 게임 exe 실행 → 플레이 가능 확인

### E-STEAM-2: 오프라인 모드

**상황**: Steam 오프라인 모드에서 실행
**기대 동작**: 게임 정상 플레이, 업적/리더보드는 온라인 복귀 시 동기화
**검증**: 인터넷 끊고 플레이 → 업적 해제 → 온라인 복귀 후 동기화 확인

### E-STEAM-3: stats.json 손상/삭제

**상황**: 통계 파일이 손상되거나 삭제된 경우
**기대 동작**: 기본값으로 초기화, 크래시 없음
**검증**: user://stats.json 삭제 후 실행 → 통계 0으로 초기화

### E-STEAM-4: 설정 파일 손상

**상황**: settings.json이 손상된 경우
**기대 동작**: 기본 설정으로 복원
**검증**: settings.json에 잘못된 JSON 작성 후 실행 → 기본값 적용

### E-STEAM-5: 해상도 극단값

**상황**: 매우 작거나 큰 해상도
**기대 동작**: UI가 깨지지 않고 표시됨
**검증**: 720p, 1080p, 1440p, 4K에서 UI 확인

## 성공/실패 판정 매트릭스

| 카테고리 | 전체 통과 | 부분 통과 | 실패 |
|---------|-----------|-----------|------|
| Steam 통합 | 초기화+오버레이+업적+리더보드+클라우드 | 초기화+오버레이+업적 | 초기화 실패 |
| 사운드 | BGM 2곡 + SFX 9종 | BGM 1곡 + SFX 5종 | 사운드 없음 |
| 스토어 | 에셋 7종+스크린샷+트레일러+설명 | 에셋+스크린샷+설명 (트레일러 미완) | 캡슐 미완 |
| 안정성 | 30분+ 크래시 0 | 30분 중 1회 비치명적 버그 | 크래시 발생 |
| 설정 | 풀스크린+해상도+볼륨+저장 | 풀스크린+볼륨 | 설정 없음 |

**전체 통과 = 출시 가능**
**부분 통과 = 해당 항목 수정 후 재검증**
**실패 = 출시 연기**
