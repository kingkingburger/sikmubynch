# SIKMUBYNCH - Steam 출시 요구사항

> Ouroboros Phase 1 | 2026-03-24 | 모호성: 15%

## 목표 (Goal)

SIKMUBYNCH(식무변처)를 2026년 하반기에 Steam Early Access로 출시한다. 스토어 페이지가 존재하고 유저가 구매 가능한 상태가 되면 성공이다.

## 제약조건 (Constraints)

- **출시 전략**: Early Access
- **목표 시기**: 2026년 하반기 (7~12월)
- **예산**: $100 (Steam 등록비만). 에셋/트레일러는 직접+AI 제작
- **지원 OS**: Windows 전용
- **입력**: 키보드 + 마우스 (컨트롤러 미지원)
- **개발자명**: 미정 (출시 전 확정 필요)
- **사업자**: 개인 명의 출시 (법인 없음)
- **사운드/음악**: AI 생성 도구(Suno/Udio 등) 활용
- **3D 모델링**: Hunyuan3D-2 로컬 (별도 Ouroboros 문서 참조)
- **스토어 에셋**: Claude가 HTML/CSS로 캡슐 이미지 목업 제작
- **트레일러**: 게임 녹화 + 편집 (도구 미정)

## 비목표 (Non-Goals)

- Full Release 품질의 완벽한 밸런싱 (EA이므로 점진적 개선)
- macOS / Linux 지원
- 컨트롤러 / Steam Deck 최적화
- DLC / 유료 확장팩 (EA 기간 내)
- 마케팅 예산 집행 (오가닉 위시리스트만)
- 트레이딩 카드 / 리치 프레즌스

## 수용 기준 (Acceptance Criteria)

1. **Steam 스토어 페이지가 공개되고 구매 가능** — 최우선 성공 기준
2. **게임이 크래시 없이 30분 이상 플레이 가능**
3. **캡슐 이미지 7종 + 스크린샷 5장+ + 트레일러 1개 준비됨**
4. **한글/영문 스토어 설명 작성됨**
5. **GodotSteam 통합 (초기화 + 오버레이 동작)**
6. **3D 모델 교체 완료 (프리미티브 → GLB)**
7. **사운드/음악 최소 1 BGM + 핵심 효과음 포함**

## Steamworks 기술 요구사항

| 항목 | 내용 |
|------|------|
| GodotSteam | Godot 4.x용 플러그인 (godotsteam.com) |
| Steam 초기화 | `Steam.steamInit()` + `Steam.run_callbacks()` |
| Export Template | GodotSteam 커스텀 템플릿 필수 |
| steam_appid.txt | 개발용 앱 ID 파일, 출시 빌드 제외 |
| 업적 | Steamworks 대시보드 정의 + GodotSteam API |
| 클라우드 세이브 | Auto-Cloud (코드 수정 없음) |
| 업로드 | SteamCMD 또는 파트너 사이트 ZIP |

- Steam 기본 EULA 사용
