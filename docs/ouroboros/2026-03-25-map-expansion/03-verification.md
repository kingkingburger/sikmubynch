# Phase 3: 검증 — 대규모 맵 확장

## 1. E2E 시나리오

### S1: 기본 게임 루프 (256x256)
- **전제:** 게임 시작, HQ at (128,128)
- **행동:** 웨이브 1 시작
- **기대:** 적이 맵 경계에서 스폰 → FlowField 따라 HQ로 이동 → 전투
- **검증:** F3 오버레이에서 enemies > 0, FPS ≥ 60

### S2: 카메라 전체 맵 이동
- **행동:** WASD로 (0,0) → (256,256) 이동, 줌 인/아웃
- **기대:** 팬이 [0,256] 범위, 줌 18~120 작동
- **검증:** 카메라가 맵 경계 밖으로 나가지 않음

### S3: 건물 배치 (전체 맵)
- **행동:** 맵 4 코너 근처에 타워 배치
- **기대:** (5,5), (250,5), (5,250), (250,250) 모두 배치 성공
- **검증:** building_grid에 4개 등록, FlowField 해당 청크 재계산

### S4: Fog of War
- **행동:** 게임 시작 → HQ 주변만 밝음 → 원거리에 타워 배치
- **기대:** HQ 시야 12셀 반경만 보임, 타워 배치 시 해당 영역 밝아짐
- **검증:** 안개 밖 영역은 어둡게 렌더링 (셰이더 vis < 0.5)

### S5: 자원 지대 활용
- **행동:** 자원 지대(48,48)에 채굴기 배치
- **기대:** 채굴기 수입 2x 보너스 적용
- **검증:** mineral/sec 증가율 비교

### S6: 미니맵
- **행동:** 우측 하단 미니맵 확인, 클릭으로 카메라 이동
- **기대:** 건물=파란점, 적=빨간점, 카메라 시야=흰 사각형
- **검증:** 미니맵 클릭 시 카메라 해당 위치로 이동

### S7: 대규모 웨이브 (1000+)
- **행동:** 웨이브 5+ 진행, 4방향 스폰
- **기대:** 1000+ 적 동시 존재, FPS ≥ 60
- **검증:** F3 오버레이 enemies ≥ 1000, FPS ≥ 60

### S8: 다방향 스폰 진행
- **행동:** 웨이브 1→5 진행
- **기대:** 웨이브 1-2: 1방향, 3-4: 2방향, 5+: 4방향
- **검증:** 미니맵에서 적 출현 방향 확인

## 2. 예외/에러 시나리오

### E1: FlowField 경로 차단
- **상황:** 건물로 HQ까지의 경로를 완전 차단
- **기대:** 적이 가장 가까운 건물 공격 (경로 없음 처리)
- **검증:** 적이 멈추지 않고 행동함

### E2: 안개 밖 건물 파괴
- **상황:** 시야 밖 바리케이드가 적에게 파괴됨
- **기대:** 이벤트 알림 표시 "건물 파괴됨!"
- **검증:** _event_label에 메시지 표시

### E3: 맵 경계 건물 배치
- **상황:** (0,0) 또는 (255,255)에 건물 배치 시도
- **기대:** 정상 배치 (유효 범위 내)
- **검증:** building_grid에 등록됨

### E4: 청크 경계 적 이동
- **상황:** 적이 청크 경계를 넘어 이동
- **기대:** 새 청크의 FlowField 방향으로 부드럽게 전환
- **검증:** 적 이동 방향에 끊김 없음

## 3. 성능 기준

| 항목 | 목표 | 측정 방법 |
|------|------|-----------|
| FPS (1000적) | ≥ 60 | F3 오버레이 |
| FlowField 글로벌 재계산 | < 5ms | 타이머 로그 |
| FlowField 청크 재계산 | < 10ms | 타이머 로그 |
| Fog 업데이트 (0.5초 간격) | < 3ms | 타이머 로그 |
| 메모리 증가 | < 50MB (vs 현재) | 태스크 매니저 |
| 빌드 시작→게임 로드 | < 5초 | 체감 |

## 4. 성공/실패 판정 매트릭스

| 카테고리 | 전체 통과 | 부분 통과 | 실패 |
|---------|-----------|-----------|------|
| E2E (S1-S8) | 8/8 pass | S1,S3,S7 pass + 나머지 | S1 or S7 fail |
| 예외 (E1-E4) | 4/4 정상 처리 | 3/4 정상 | E1 미처리 (적 멈춤) |
